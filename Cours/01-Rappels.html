A l'interieur d'un paragraphe, on peut retrouver des balises de "texte" : le span : ne fait rien, juste pour encadrer un bout de texte, strong : met du texte en gras, + mots clefs pour le SEO, b pour bold : texte en gras u : underline, pour souligner, a : ancre, pour les liens, mark : pour surligner, i, italic , em , emphase : mots clefs pour le SEO ( + italique ) Sup, supérieur pour mettre en exposant, Sub, pour mettre en indice, S pour barrer du texte Q ou quote, citation dans le texte,

Div : division, pas de valeur sémantique, assez peu de style par défaut, servent essentiellement à diviser ou encadrer des blocks pour le style
Entête de page : - Balise encadrante, avec une valeur sémantique - 2 rôles : - Soit, à la racine, dans ce cas, entête de la page, unique - Soit on s'en sert pour découper des blocks de contenu,
- Contenu principal de la page -> Ce que l'user est venu chercher -> Contenu unique, -> Une seule balise main visible par page
- Contenu secondaire ou complémentaire - éléments, ou des liens en rapport avec le main
- Groupe de liens de navigation - navbar / menu
- Pied de page, -> Soit pour la page en elle même -> Soit le footer d'un article, d'une carte, d'un contenu...
